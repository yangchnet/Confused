# 最小的k个数

## 1. 问题描述
输入整数数组 arr ，找出其中最小的 k 个数。例如，输入4、5、1、6、2、7、3、8这8个数字，则最小的4个数字是1、2、3、4。

## 2. 例子
```
输入：arr = [3,2,1], k = 2
输出：[1,2] 或者 [2,1]
```

```
输入：arr = [0,1,2,1], k = 1
输出：[0]
```

## 3. 解题思路
输入：4、5、1、6、2、7、3、8
```
        minNums               max
init:   [4, 5, 1, 6]           6
2       [4, 5, 1, 2]           5
7       [4, 5, 1, 2]           5
3       [4, 3, 1, 2]           4
8       [4, 3, 1, 2]           4
result : [4, 3, 1, 2]
```
维护minNums数组和max  
初始化：复制所给数组的前k个到minNums,并设其中最大值为max  
从give[k]开始遍历数组  
若give[k]小于max，则替换掉minNums中的最大值，  
若give[k]大于max, 则k++  
直到遍历结束
