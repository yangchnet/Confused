# 把数组排成最小的数

## 1. 题目描述
输入一个非负整数数组，把数组里所有数字拼接起来排成一个数，打印能拼接出的所有数字中最小的一个。

## 2. 示例
```
输入: [10,2]
输出: "102"
```

```
输入: [3,30,34,5,9]
输出: "3033459"
```

## 3. 解题
设x, y为两个数字对应的字符串，则
若[x, y] > [y, x], 则应将y排列在x的前面, 即Less(x, y) = false
若[x, y] < [y, x], 则应将x排列在y的前面, 即Less(x, y) = true

根据这个关系，可利用sort.Sort()接口，通过自定义Less(i, j int)bool 函数，即可完成全部数字的排序
