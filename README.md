# 在Leetcode上的剑指offer刷题记录

## 1. 说明
刷题时先用了Java语言，后面全部转为golang。 
每个题目为一个文件夹，文件夹内有des.md和源代码文件。des.md为对题目的描述及记录的解题方法。
有的题目后面有f, s, m, d等标识，其含义为
|标记|含义|
|-----|-----|
|f|错误（或未通过）|
|s|题目难度：简单|
|m|题目难度：中等|
|d|题目难度：困难|
